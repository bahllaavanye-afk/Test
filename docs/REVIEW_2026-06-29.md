## A. Infra / Website reliability
- **[P0] Render free tier sleeps** — `/health` returned `000` (cold start). While asleep the in-app
  employees (StrategyRunner / PriceFeed / ModelingEngineer / AlgoAgent) halt, so "24/7" is not real.
  Fix: paid Render ($7/mo) **or** an external uptime pinger (UptimeRobot is more reliable than GH cron).
- **[P1] LLM cascade intermittently unavailable** — AI PR reviews skipped twice this session
  ("cascade unavailable"). CONTINUITY says only Groq's free key works live. Refresh dead free-provider
  keys in Doppler (SambaNova/Together/Hyperbolic) + fund the Anthropic backstop.
- **[P1] Slack MCP/token flaky** — couldn't read channels this session. Refresh the Slack token; the
  desk depends on it for alerts/visibility.

## B. Multi-agent system (the "employees")
- **[P1] 86 workflows — audit & consolidate.** Many near-duplicates (`slack-*`, `agent-*`, `render-fix-*`).
  Build an **employee manifest** (employee → owner workflow → cadence → last-success) and dedupe to cut
  cost/noise and make health legible.
- **[P1] No single employee-health board.** `agent-health-*` / `system-status` exist — verify they
  actually **page** when an employee hasn't succeeded in N hours; make the smoke test a hard gate
  (IMPROVEMENTS: "agent smoke test red but unwatched").
- **[P1] Self-improvement isn't reward-gated.** `continuous_improver` commits directly to `main`;
  route its changes through `reward-gate.yml` (verifiable improvement) + draft PR + tests, not direct commit.
- **[P2] ML employees inert in prod.** 23 ML modules skip on missing torch/xgboost on Render. Either run
  them on a worker with the `[ml]` extra, or mark them clearly degraded so they're not counted as "working."

## C. Desks / alpha coverage
- **[P0] Forex & Commodities desks absent** — 0 strategies, no `market_type`, no desk in the taxonomy,
  no data routing, no scheduled desk workflow. Build: forex (carry, trend/momentum) + commodity
  (term-structure roll, momentum, gold/oil mean-reversion) strategies → register → add Forex/Commodities
  to `_MARKET_TYPE_DESK` → route data_loader symbols (`EURUSD=X`, `GC=F`, `CL=F`) → add a scheduled desk.
- **[P1] Crypto live quotes still via geo-blocked Binance.** Backtest/data path now uses Alpaca (#258);
  wire Alpaca crypto into `price_feed` for live quotes too.
- **[P2] Options desk** research-complete but not in the Bot builder (IMPROVEMENTS Stage 2/3).

## D. Code quality / tests
- **[P1] 435 broad `except Exception`** — narrow + log; blanket catches hide agent failures and make

- **[P1] Stale provider model IDs** in `llm_common` (Cerebras/NVIDIA) — audit endpoints/models.
- **[P2] Deprecations** — pytest-asyncio `event_loop_policy` fixture, Starlette `TestClient`+httpx.

## E. Observability
- **[P1] No tracing/memory** — add Langfuse/OTel on `llm_common`; pgvector (Mem0/Letta) durable memory.
- **[P1] VaR threshold + Sharpe-shortfall** flagged in `#leadership-summary`/`#okrs` — wire risk/OKR
